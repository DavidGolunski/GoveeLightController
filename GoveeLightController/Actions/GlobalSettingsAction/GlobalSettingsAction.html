Setup Instructions

Set up the default devices that are controlled by the plugin here:
  • Turn on "LAN Control" for each light you want to control (you can do this using the Govee mobile app).
  • Your Govee devices need to be connected to the same network as the StreamDeck.
  • Your devices need to be able to communicate with each other in the network (this is usually the default in the router settings).
  • Recommended: Give your devices a static IP inside the router settings.

Button Press

Scan for available devices and automatically populate the "Device IP List". Don't forget to save if you want to keep the automatically found addresses
